How It Works

This demo showcases FHE16-CNN, a convolutional neural network that performs inference on encrypted data using Fully Homomorphic Encryption (FHE).

  • Privacy: Your input data is encrypted before processing - the server never sees the actual pixel values
  • Security: Uses 16-bit FHE scheme with lattice-based cryptography
  • Architecture: Conv(3√ó3, stride=3) ‚Üí ReLU ‚Üí SumPool(3√ó3) ‚Üí FC(27‚Üí10)
  • Accuracy: ~97% on MNIST test set
