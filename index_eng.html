About Us

WaLLLnut is a lab-based startup team from the Cryptographic Communications Laboratory at Hanyang University. Based on quantum-resistant Fully Homomorphic Encryption (FHE) and Secure Multi-Party Computation (MPC), we ensure the security and privacy of data, computations, and control in AI, robotics, and cloud environments.

🧠 WaLLLnut Technology

⚙️ FHE16 – High-Speed Homomorphic Encryption Library

  • High-speed FHE structure for real-time AI and robot control
  • Compressed key switching and optimized modularization to reduce communication bottlenecks
  • Versatile and ready-to-use for privacy-preserving tech like PIR, LLM inference, federated learning, etc.

🔐 Secure Multi-Party Computation (MPC)

  • Compute collaboratively while keeping data ownership intact
  • Secure computation even in zero-trust environments between companies or individuals
  • Combine with FHE for both security and practicality
  • Planned expansion to FHE-based hybrid MPC engines

☁️ Cloud-Based FHE Services

  • Directly invoke homomorphic encryption computation via the FHE16 API
  • Apply AI models and statistical analysis without decrypting data
  • Security computation platform that provides insights without exposing sensitive information

🔎 Private Information Retrieval (PIR)

  • Query only the necessary data without revealing what is being requested
  • Even the server cannot know what the user is asking for
  • Guarantees complete privacy in recommender systems, medical data searches, secure search engines, etc.
  • Equipped with high-performance FHE-based PIR engine

📜 Smart Contracts

WaLLLnut’s cryptographic technology goes beyond protecting data. It also enables secure execution and automation of smart contracts.

By using FHE and MPC, contract terms are processed in encrypted form, and only the results are revealed—enabling trustworthy execution without centralized authorities.

This is highly useful in finance, automated legal contracts, and blockchain-based services that require strong security and privacy.

📘 FHE and MPC Made Simple

🔐 FHE (Fully Homomorphic Encryption)

A technology that enables computation on encrypted data without decryption.
This allows analysis and AI processing while keeping sensitive information safe.

👥 MPC (Multi-Party Computation)

A structure where multiple parties compute collaboratively without revealing their data.
Enables cooperation in zero-trust environments.
