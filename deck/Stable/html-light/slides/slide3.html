문제 정의

블록체인 투명성 vs 기업 기밀성

현재 문제
거래 금액 완전 노출
모든 Stablecoin 이체 금액이 온체인 공개
→ Etherscan/Explorer에서 실시간 조회 가능
거래 상대방 추적
지갑 주소 기반 거래 네트워크 분석 가능
→ Chainalysis, Nansen 등 이미 상용화
기업 전략 정보 유출
M&A, 투자, 파트너십 정보 경쟁사에 노출
→ 대규모 거래 시 시장 교란 가능
규제 요구사항
자본시장법 제117조 (정보의 관리)
STO = 토큰증권 = 금융투자상품
→ 코스콤/증권사는 투자자 거래정보를 제3자에 공개 금지
⚠ 블록체인 공개 = 117조 위반 소지
특정금융정보법 제5조의2 (AML/KYC)
감독기관은 의심거래 조회 권한 필요
→ 완전 익명화 불가, 감사 접근권 보장 필수
전자금융거래법 제21조 (기록보존)
5년간 거래기록 보존 및 감독기관 제출 의무
→ 암호화된 상태로도 검증 가능해야 함
핵심 딜레마
기업 요구 거래 정보 비공개
vs
규제 요구 감사 접근권 보장
→ LatticA: 암호화 상태로 감사 수행 = 양립 가능
자본시장법 전문 특정금융정보법 Chainalysis Compliance
