01 Executive Summary

한 줄 요약: 블라인드 경매 + 분쟁 없는 증빙 시스템

AgriBlind는 도매시장 경매에서
"응찰자 비공개(블라인드)"를 시스템으로 강제하고,
결과의 정당성을 '검증 가능한 증빙 패키지'로
자동 생성해 분쟁·정정·민원을 줄이는
Web2 솔루션

! 2026년 즉시 매출 가능: 정부가 옴부즈만·블라인드 경매 확대/의무화·대금정산조직 도입을 추진 중

경매 조작 시비 원천 차단
분쟁 처리 시간 90% 단축

최고가격으로 낙찰 후 낙찰자/낙찰가격 공개

  • 블라인드 경매 확대/의무화

  • 전자경매 투명성 강화 과제
  • 담합/불공정 기사·제재 사례
  • 정책이 괜히 나온 게 아님

06 FHE16 Application

FHE16은 "여기서" 필요하다 — latency 허용 구간에만

! "무조건 FHE"가 아니라, 'FHE가 필요한 지점'에만 적용해야 사업성
1

블라인드 "최고가 결정" 증명

단순 "ID 숨김"만이면 차별화 약함

  • 응찰값을 암호화해 제출
  • 서버는 암호문 상태로 비교/최대값 계산
  • optimistic proof로 이의제기 시 검증 강화

경매는 로트 단위라 수백ms~수초 허용 가능

2

대금정산조직 네팅/한도 계산

참여자 거래·포지션 노출 시 참여 꺼림

  • 순정산액/한도 초과/담보요구만 공개
  • 원본 거래 데이터는 비가시
  • 온라인 확대 정책 목표와 정합

스테이블코인 정산 레일까지 연결 가능

07 Value Proposition

고객 가치: B(시간) + A(비용)으로 먼저 증명 → C(수익)로 확장

B. 시간 단축

가장 설득 쉬움

  • 분쟁 발생 시 "자료 수집" → 자동 증빙
  • 처리 리드타임: 일/주 → 시간 단위
  • 옴부즈만/위원회 도입 시 즉시 필요

A. 비용 절감

정산/분쟁/정정 비용

  • 운영 인력 투입 감소
  • 판매원표 정정 처리 비용↓
  • 법적 분쟁/민원 대응 비용↓

C. 거래량 증가

신뢰 → 참여 확대

  • 온라인 판매자 149→1,136 급증
  • 신뢰가 참여 장벽 낮춤
  • 거래액 성장 레버리지

온라인도매시장 실적: 유통비용률 7.4%p 감소 — 정부가 '비용 절감'에 돈 쓰는 명분이 이미 존재
