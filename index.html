"AI와 로봇의 시대, 데이터 연산과 시스템 제어는 보호받으며 수행되어야 합니다."

WaLLLnut은?

한양대학교 암호통신연구실에서 시작된 실험실 창업팀으로, 양자내성을 가진 동형암호(FHE)와 다자간 연산(MPC) 기술을 기반으로 AI, 로보틱스, 클라우드 환경에서의 데이터, 연산 및 제어의 보안과 프라이버시를 보장합니다.

이론만 있는 암호는 실제 상황에서 쓸 수 없습니다.
WaLLLnut은 실제로 작동하고, 현실에서 필요한 속도와 안전성을 갖춘 "현장 중심 양자내성을 가진 다자간 동형암호 기술과 서비스”를 만듭니다.
보안성과 효율성의 균형을 조절하는 차세대 암호 시스템을 설계합니다.
연구를 넘어 산업으로 이어지는 실용 암호기술을 추구합니다.

🧠 WaLLLnut의 기술

⚙️ FHE16 – 고속 동형암호 라이브러리

  • 실시간 AI·로봇 제어를 위한 고속 FHE 연산 구조
  • 통신 병목을 줄이는 압축 키 스위칭 및 최적화된 모듈화
  • 다양한 프라이버시 보호 기술에 즉시 적용 가능한 범용성 (예: PIR, LLM inference, federated learning 등)

🔐 Secure Multi-Party Computation (MPC)

  • 데이터 소유권을 지키며 협업 가능한 연산 구조
  • 기업 간, 개인 간 제로 트러스트 환경에서의 보안 연산 가능
  • FHE와 결합해 연산 보안성과 실용성을 모두 확보
  • 향후 동형암호 기반 MPC 하이브리드 엔진으로 확장 예정

☁️ 클라우드 동형암호 서비스

  • FHE16 API를 통해 동형암호 연산을 클라우드에서 바로 호출 가능
  • 데이터 복호화 없이 다양한 AI 모델과 통계 분석 적용 가능
  • 민감 정보 유출 없이, 필요한 분석 결과만 안전하게 도출하는 보안 연산 플랫폼

🔎 Private Information Retrieval (PIR)

  • 불필요한 정보를 요청하지 않고, 원하는 데이터만 쿼리하는 구조
  • 사용자가 어떤 데이터를 요청했는지 서버조차 알 수 없는 query 기술
  • 추천 시스템, 의료정보 검색, 보안 검색엔진 등에서의 완전한 프라이버시 보장
  • FHE 기반으로 구현된 고성능 PIR 엔진 탑재

📜 스마트 컨트랙트

WaLLLnut의 암호 기술은 단순히 데이터를 보호하는 것을 넘어서, 스마트 컨트랙트(smart contract)와도 결합되어 실행 보안성과 계약 자동화를 동시에 제공합니다.

FHE와 MPC를 활용하면 계약 내용은 암호화된 채로 처리되고, 실행 결과만 공개되기 때문에 중앙기관 없이도 신뢰할 수 있는 계약 실행이 가능합니다.

이는 특히 금융 거래, 자동화된 법적 계약, 블록체인 기반 서비스에서 높은 보안성과 개인정보 보호를 요구할 때 매우 유용합니다.

📘 FHE와 MPC, 쉽게 이해하기

🔐 FHE (Fully Homomorphic Encryption)

데이터를 암호화한 상태 그대로 연산할 수 있는 기술입니다.
복호화하지 않아도 분석이나 AI 모델 처리가 가능하기 때문에, 민감한 정보를 안전하게 다룰 수 있습니다.

👥 MPC (Multi-Party Computation)

여러 주체가 데이터를 공개하지 않고도 함께 계산하고 결과만 공유할 수 있는 구조입니다.
서로를 신뢰하지 않는 제로 트러스트 환경에서도 협력 연산이 가능합니다.

💡 WaLLLnut 기술의 응용 예시

WaLLLnut의 기술은 현실 세계의 다양한 분야에서 개인 정보 보호와 데이터 보안이라는 중요한 문제를 해결합니다.

🤖 로봇

  • 로봇이 사용자의 위치나 행동 패턴을 복호화 없이 인식·판단하여 사생활을 침해하지 않고 작동할 수 있습니다.
  • 예: AI 도우미 로봇이 집 안 환경을 인식하면서도 개인 정보를 서버에 노출하지 않음

💰 금융

  • 은행과 핀테크 기업들이 고객의 민감한 자산 정보를 복호화하지 않고 신용 분석·위험 평가를 수행할 수 있습니다.
  • 예: 고객의 소득·지출 내역을 비공개 상태로 신용점수 산정

🧠 AI

  • 사용자의 데이터를 노출하지 않고 AI 모델이 연산을 수행합니다. FHE 기반 LLM 추론도 가능해집니다.
  • 예: 사용자의 감정·취향 분석을 서버에 전달하지 않고 모델이 동형암호 상태에서 처리

📄 계약 및 블록체인

  • 스마트 컨트랙트와 결합하여 거래 내역이나 계약 조건이 민감한 경우에도 보안성이 보장된 자동 계약 이행이 가능합니다.
  • 예: 의료 계약이나 고용 계약의 민감 데이터를 암호화된 상태에서 검증 및 이행

🧬 의료

  • 환자의 진단 기록이나 유전자 정보 등을 병원 간 공유하거나 분석할 때도 프라이버시를 완벽하게 보호할 수 있습니다.
  • 예: 병원 A의 환자 데이터를 병원 B의 AI가 분석하되 환자 정보는 유출되지 않음

🗂 데이터베이스 / 검색

  • 서버는 사용자가 어떤 정보를 찾는지 모른 채 정확한 데이터를 제공할 수 있습니다.
  • 예: PIR 기반 검색엔진 – 검색 기록이 서버에 남지 않는 완전한 비공개 검색
